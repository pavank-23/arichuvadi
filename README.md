## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

An application to help beginners learn Tamil. Use the table learn about alphabets, their pronunciations, and increase your vocabulary with the suggested words. Users can also use the inbuilt translator to translate new phrases they come across.

### Installation

1. Get a free API Keys for translation from [https://rapidapi.com/googlecloud/api/google-translate1](https://rapidapi.com/googlecloud/api/google-translate1)
2. Clone the repo
   ```sh
   git clone https://github.com/pavank-23/arichuvadi.git
   ```
3. Install NPM packages
   ```sh
   npm install
   npm run dev
   ```
